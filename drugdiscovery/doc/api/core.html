drugdiscovery.core¶

class Engine(task, train_set, valid_set, test_set, optimizer, gpus=None, batch_size=1, log_interval=100, checkpoint_interval=2)[source]¶

General class that handles everything about training and test of a task.

This class performs synchronous distributed parallel training over multiple nodes and GPUs. To invoke with multiple GPUs,

python -m torch.distributed.launch --nproc_per_node={number_of_gpus} {your_script.py}
Parameters:
  • task (nn.Module) – task
  • train_set (data.Dataset) – training set
  • valid_set (data.Dataset) – validation set
  • test_set (data.Dataset) – test set
  • optimizer (optim.Optimizer) – optimizer
  • gpus (list of int, optional) – GPUs. By default, all GPUs will be used.
  • batch_size (int, optional) – batch size
  • log_interval (int, optional) – log every n batches
  • checkpoint_interval (int, optional) – checkpoint every n epochs
